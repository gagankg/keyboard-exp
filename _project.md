# keyboard-exp

## What It Is
A touch typing test app — replica of typr.aswin.fyi — built with the keyb keyboard component library.

## Stack
Next.js 16 (App Router), TypeScript, Tailwind CSS v4, shadcn/ui, keyb (himan.me), JetBrains Mono

## Goal
Faithful replica of typr.aswin.fyi with: duration selector (15/30/60s), live timer, mistakes + wrong-word tracking, character-level text highlighting, visual keyboard that lights up as you type, theme switcher, dark/light/auto mode.
